# Copy & Voice — تطبيق المعارض

> **المهمة**: نصوص جاهزة للنسخ-اللصق، متوافقة مع الثقافة السعودية المهنية.
> **اللغة الأساسية**: عربي فصحى مبسطة (MSA لايت) — ليست كلاسيكية، ليست عامية، تشبه نشرة Aramco الداخلية أو محادثة في لوبي فندق Four Seasons في الرياض.
> **مكمّل لـ**: [`08-brand-guidelines.md`](./08-brand-guidelines.md)

---

## 1. مبادئ صوت العلامة (Voice Principles)

### الـ 5 قواعد الذهبية للنص

1. **الإنسان قبل النظام** — "أرسلت طلبك" ✅ ، ليس "تم إرسال الطلب بنجاح" ❌.
2. **الرقم قبل الصفة** — "خلال 24 ساعة" ✅ ، ليس "بسرعة" ❌.
3. **الفعل قبل التسويق** — "احفظ" / "أرسل" / "وافق" ✅ ، ليس "تميّز / استمتع / اكتشف" ❌.
4. **الشفافية قبل التلميع** — "Admin يقرأ هذه المحادثة" ✅ ، ليس "محادثة آمنة ومحمية" ❌.
5. **الاحترام قبل الود الزائف** — لا "عميلنا الكريم" ولا "أهلاً يا غالي" — اسم المستخدم مباشرة.

### نسبة العربي/الإنجليزي
- **العربي افتراضي 100%**.
- المصطلحات التقنية المعروفة دولياً تبقى لاتينية: `RFQ`, `PDF`, `ZATCA`, `CR`, `VAT`, `ID`, `KPI`, `ROI`, `Admin`.
- لا نترجم "Dashboard" — نقول "لوحة التحكم" (مقبول)، أو نتركها "Dashboard" في المسار `/dashboard`.

### مَن نتجنبه

❌ **لغة الرومانسية الإعلانية**: "رحلتك تبدأ هنا" / "اكتشف عالمنا" / "نحقق أحلامك"
❌ **لغة الكلاسيكية الفصحى الثقيلة**: "نحرص على تقديم أرقى الخدمات" / "إذ نسعى جاهدين"
❌ **لغة الدعاية الفجة**: "الأفضل في المملكة" / "رقم 1 بلا منازع" (لا دليل = ادعاء)
❌ **العامية الخليجية الواضحة**: "ولّع طلبك" / "يا الغالي" / "حياك" — غير مهني للـ B2B
❌ **العامية المصرية**: "خلاص" / "يلا بينا" — لا تناسب الجمهور السعودي

## 2. الكلمات الأساسية (Glossary — Use These Exactly)

| المفهوم | الكلمة المعتمدة | لا تستخدم |
|--------|-----------------|----------|
| الطلب | **طلب عرض سعر / RFQ** | استفسار، بحث |
| العرض من المورد | **عرض** | بروبوزال، اقتراح |
| المورد | **مورد** | بائع، spar، vendor |
| العميل | **شركة** أو **عميل** | مشتري، زبون |
| الضمان المالي | **Escrow / ضمان** | إيداع، حفظ |
| الموافقة | **اعتماد** | تأكيد، إقرار |
| التسليم | **تسليم** | توريد، تسليم نهائي |
| التقييم | **تقييم** | مراجعة، رأي |
| الإدارة | **Admin** أو **فريق المنصة** | المسؤول، المدير |
| الفاتورة | **فاتورة ضريبية** | فاتورة، invoice |
| العمولة | **عمولة 5%** (دائماً مع الرقم) | رسوم، نسبة |
| الفزعة | **زر الفزعة** | تنبيه طارئ، help |


#### Value Props — الميزات الـ 9

| العنوان (مختصر) | الشرح في سطر |
|------------------|---------------|
| **عمولة 5% فقط** | بدل 40-60% للوكالات. شفافية كاملة، بدون رسوم خفية. |
| **ضمان مالي (Escrow)** | فلوسك في حساب المنصة. لا تتحرك للمورد إلا بعد توقيعك على التسليم. |
| **AI يقارن العروض** | جدول مقارنة ذكي + توصية مبنية على السعر، الجودة، والتقييمات. |
| **زر الفزعة** | في كل شاشة مشروع. اضغط، يصلك Admin خلال 30 ثانية. |
| **توثيق الاتفاق بـ AI** | كل طرف يكتب فهمه، AI يكشف التناقضات قبل بدء العمل. |
| **Admin طرف ثالث صامت** | يستمع لكل محادثة، يتدخل لو لزم. لا مفاجآت. |
| **فاتورة ضريبية موحدة** | فاتورة ZATCA من المنصة، مهما تعددت الموردين. |
| **CRM مجاني للموردين** | إدارة عملاء، تقارير، Pipeline. مجاناً، لأن نجاحهم نجاحنا. |
| **عين الـ CEO** | رابط Read-only لـ CEO لمتابعة كل المعاملات بدون تدخل. |

#### How It Works (A2) — الـ 7 خطوات

```
1. أرسل طلبك (RFQ)
   املأ نموذج بسيط حسب الخدمة المطلوبة. نحدد الموردين المؤهلين تلقائياً.

2. استقبل العروض خلال 24 ساعة
   3 موردين معتمدين على الأقل سيرسلون عروضهم.

3. قارن بمساعدة AI
   جدول مقارنة + توصية ذكية. اختر حتى 4 موردين للتفاوض.

4. تفاوض في الشات
   Real-time مع كل مورد على حدة. Admin يستمع، يتدخل لو احتجت.

5. وثّق الاتفاق
   كل طرف يكتب فهمه. AI يكشف الفجوات. Admin يعتمد. لا مفاجآت لاحقاً.

6. حوّل 50% (Escrow)
   فلوسك في حساب المنصة. المورد يبدأ. أنت مطمئن.

7. وافق على التسليم → نحوّل للمورد
   عند رضاك، نحوّل للمورد بعد خصم 5% فقط. إنتهت المعاملة.
```

#### Pricing Page (A10)

```
العنوان:
عمولة واحدة. شفافة. 5%.

الشرح:
2% من العميل (مضافة فوق سعر العرض)
3% من المورد (مخصومة من حصته)

ما تحصل عليه مقابلها:
✓ Escrow على كل ريال
✓ AI لتحليل العروض والاتفاقيات
✓ زر الفزعة + تدخل Admin مجاني
✓ فاتورة ضريبية موحدة (ZATCA)
✓ تخزين كل ملفات المشروع للأبد
✓ CRM مجاني للموردين

مقارنة مع الوكالات التقليدية:
الوكالة → 40-60% هامش مخفي
نحن → 5% واضحة على الفاتورة
```

## 12. التحقق النهائي قبل النشر (Copy QA Checklist)

قبل أي نص يذهب للـ Production:

- [ ] الأرقام `dir="ltr"` حتى في RTL.
- [ ] لا يوجد "نحن نسعى لـ..." أو لغة كلاسيكية ثقيلة.
- [ ] الـ CTA فعل واضح (ليس "Click here" أو "تعرف أكثر").
- [ ] الإشعار يقول: ماذا حدث + ماذا تفعل.
- [ ] الخطأ يقول: ما المشكلة + الحل المباشر.
- [ ] لا "Lorem Ipsum" متبقي.
- [ ] الأسماء (شركات، أشخاص) في الـ examples تبدو سعودية واقعية.
- [ ] لا emoji decorative في UI core (مسموح فقط في 3 مواضع: تأكيد إنجاز كبير، Wow Moment، Panic warning).
- [ ] العملة: **﷼** (رمز الريال السعودي الجديد) ليس "SAR" أو "ر.س.".
- [ ] التاريخ الميلادي افتراضي، الهجري متاح كـ secondary فقط لو طلب المستخدم.
- [ ] لا اعتذار غامض ("نأسف للإزعاج") — اعتذر عن شيء محدد لو حدث.
- [ ] متناسق مع `08-brand-guidelines.md` Section 12 (Voice).
